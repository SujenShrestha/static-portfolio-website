Research

Logo
The logo on the left side is from a sebsite named "Dr Steven Kinnear". What I really liked about this design is its shear simplicity. It does not use any stylish fonts or images, it only contains simple text. Yet it is such a good design. This simplicity inspired me to create a similar logo without any images or sophisticated design. Thus, this lead me to create a simple text logo for this website.

Social Media Icons
The navigation on the left side is from a website named "Robby Leonardi". I liked the design of the layout and also circular social media icons. The icons are placed in the right side of the bar in equidistance with each other. I really liked this design and because of this I attempted to recreate a similar design for my website. So, I created this circular social media icons which redirects to their respective page when clicked on it.

Navigation Bar
The navigation on the left side is from a website named "Dr Steven Kinnear". This is another reference that I took from this website. I liked the navigation design which changes colour when hovered over. I found various fancy designs in other websites, some of them performed animation effects, drop-box effects, etc. But I thought this design would be perfect for a simple website like mine. So, I changed the design a little by placing the navigation pane to the left side of bar so that people can easily navigate through different pages within the website.

Contact page
The navigation on the left side is from a website named "MetaLearn". The suitably annotated boxes and the layout has a very friendly look. I found this simple design very elegant, so I recreated a similar form and placed it in the contact page so that anyone who is new to the site can find it easily and convey their message without much trouble. Also, in case if they leave the fields empty then a dialogue box will appear stating the fields are empty else it will give a confirmation message when message is received.

Footer
The navigation on the left side is from a website named "Andy Martin". The footer section in this website really caught my eye with its features. It contains a naviagtion bar, social media redirect page icons and a copyright notice. I think that providing all these functionality at the bottom really improves the users ability to browse through all the related pages within the website significantly easier and beetter. Thus, I created a similar design providing the navigation tools, social media icons and copyright notice.
